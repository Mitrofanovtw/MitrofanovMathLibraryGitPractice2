Методы класса Calculator
- Add(double a, double b): Сложение двух чисел.
- Subtract(double a, double b): Вычитание.
- Multiply(double a, double b): Умножение.
- Divide(double a, double b): Деление с обработкой на 0.
- IsPrime(int number): Проверка на простоту.
- Power(double number, double power): Возведение в степень.
- Factorial(int n): Факториал.
- SolveQuadratic(double a, double b, double c, out double? x1, out double? x2): Решение квадратного уравнения.
